# 嘴替移动端页面访问说明

## 启动方式

在项目目录 `zuiti/` 下执行：

```bash
npm install
npm run dev
```

启动后访问：

```text
http://localhost:3000/
```

## 页面路由

| 页面 | 路由 | 用途 |
| --- | --- | --- |
| 首页 | `/` | 快速选择沟通场景，进入表达转换流程。 |
| 输入页 | `/input` | 输入真实想法，并从 6 种风格卡片中选择表达策略。 |
| 语气仪表盘 | `/tone` | 预览表达，并通过礼貌程度、正式程度、关系距离控制语气。 |
| 结果页 | `/results` | 展示微信短句版、邮件正式版、当面沟通版三种结果。 |

## 设计尺寸

页面按 iPhone SE 展示尺寸制作：

```text
375 x 750
```

桌面浏览器访问时，移动端页面会居中展示；使用浏览器设备模式设置为 `375x750` 可查看目标效果。
